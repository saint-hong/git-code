# git code summary..

## 1. basic
### 1) move
```
$ cd <dir name> : <dir name> 으로 이동
$ cd /c/<dir name> : 절대경로 /c/<dir name> 으로 이동
$ cd . : 현재 디렉토리
$ cd .. : 상위 디렉토리 이동
$ cd ../../ : 두번쨰 상위 디렉토리 이동
```
### 2) confirm
```
$ pwd : 현재 위치 확인
$ ls : 현재 디렉토리 안의 디렉토리와 파일을 확인
$ ls -al : 현재 디렉토리 안의 디렉토리와 파일의 chmod, 생성일시 등의 정보를 확인
	   (숨겨진 파일, 폴더도 확인 가능)
```
